Domain Intelligence API

Endpoints

GET
/v1/rdap/{domain}
Registration data via IANA RDAP bootstrap. Replaces WHOIS.
GET
/v1/dns/{domain}
DNS records over HTTPS. A, AAAA, NS, MX, CNAME, SOA, TXT, CAA.
GET
/v1/certificates/{domain}
Certificate Transparency log search via crt.sh. Find every cert ever issued.
POST
/v1/bulk
Look up to 20 domains in parallel. Mix RDAP, DNS, and cert queries.

Quick start — no account needed

# Health check
curl https://domainintel.onrender.com/v1/health
→ {"status":"ok"}

# RDAP lookup
curl https://domainintel.onrender.com/v1/rdap/example.com
→ {"domain":"example.com","status":"active","nameservers":[...]}

# DNS A records
curl https://domainintel.onrender.com/v1/dns/github.com?type=A
→ {"domain":"github.com","records":[{"type":"A","value":"140.82.121.3"}]}

# Certificates
curl https://domainintel.onrender.com/v1/certificates/github.com?limit=3
→ {"domain":"github.com","certificates":[...]}

# Bulk — 20 domains in parallel
curl -X POST https://domainintel.onrender.com/v1/bulk \
  -H 'Content-Type: application/json' \
  -d '{"domains":["example.com","google.com"],"types":["rdap","dns"]}'

IANA RDAP Bootstrap

No hardcoded registry list. Automatically routes to the correct RDAP server for every TLD.

DNS over HTTPS

Encrypted DNS lookups via Cloudflare and Google resolvers. No plaintext UDP to port 53.

Certificate Transparency

Query crt.sh logs. Find subdomains, pre-issuance certs, and expired certificates.

Bulk Parallel Lookups

Fire up to 20 domains in a single request. Returns results as they arrive — no sequential bottleneck.
